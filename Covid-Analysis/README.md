# COVID-19 Data Analysis Project

This project performs exploratory data analysis (EDA) on global COVID-19 data to understand the spread, death rate, recovery rate, and country-wise trends.

## Dataset
Download from:  
https://ourworldindata.org/covid-deaths

Save the file as **covid.csv** inside this folder.

## What This Project Does
- Cleans and preprocesses COVID dataset  
- Analyzes confirmed, recovered, and death counts  
- Calculates death rate and recovery rate  
- Finds most affected countries  
- Visualizes trends using Matplotlib  

## How to Run
```bash
python covid.py
